## Vue Component

- 컴포넌트 사용 2단계
  
  - 컴포넌트 파일 생성
    
    - MyComponent.vue 생성
  - 컴포넌트 등록 (import)
    
    - App 컴포넌트에 MyComponent 를 등록
      
    - App(부모) → MyComponent(자식) 관계 형성
      

### MyComponentItem 컴포넌트 등록 후 활용

- MyComponentItem은 MyComponent의 자식 컴포넌트
  
- 컴포넌트의 재사용성 확인
